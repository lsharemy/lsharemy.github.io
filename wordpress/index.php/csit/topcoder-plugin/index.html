Topcoder插件安装 | Topcoder plug-ins - KawigiEdit & FileEdit + CodeProcessor + TZTester

重装了系统，又要装插件，趁这个机会就总结下吧。 官方的文档在这里：http://apps.topcoder.com/wiki/display/tc/How+to+install+The+Arena+plug-ins

用的比较多就应该是KawigiEdit和FileEdit + CodeProcessor + TZTester组合吧 首先是KawigiEdit，相当于一个增强的编辑器，其中也包括了代码生成等功能，安装过程和功能特性插件页面都有，就不累赘了，下面是安装好之后的效果（还有即时记分功能噢！）：

有一点需要注意，就是要用这个插件的测试功能，必须要有g++，在windows下的话就要安装一个mingw了，我装的是CodeBlocks，里面带有mingw，把bin目录加一下环境变量就ok了。 然后就是FileEdit + CodeProcessor + TZTester组合，其中FileEdit让你可以在一个外部文件中编辑代码，然后在TC里直接提交（是不是很酷，不过提交之前别忘了先保存下外面的文件）。CodeProcessor则是一个代码处理插件，它允许你自己写一个类来对代码进行处理，然后再交给编辑器，等编辑完代码，需要提交的时候，代码也会经过它处理一下，才交给平台。所以她本身不能独立工作，需要一个处理类以及一个编辑器，这个类也有人给我们写了一个，TZTester，用来生成测试代码，而编辑器则可以用FileEdit、PopsEdit 或标准的编辑器，我选择的FileEdit，因为目前还离不开IDE。 安装过程在上面第一个链接中有，也不累赘啦。 需要注意的几点，FileEdit安装好之后，需要改一下代码输出目录（桌面比较方便），然后还需要改下代码模板。默认的代码模板为：

$BEGINCUT$
$PROBLEMDESC$
$ENDCUT$

#line $NEXTLINENUMBER$ “$FILENAME$”

#include <vector>

#include <string>

class $CLASSNAME$ {
public:
$RC$ $METHODNAME$($METHODPARMS$) {

}

};
我一般改成这样：
