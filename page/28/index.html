JM代码阅读之三帧内预测 | JM Code Notes 3 – Intra Prediction

帧内预测依据先前已经编码并重建好的块（左上、上、右上、左）形成一个预测块P，当前块减去这个预测块，将差值进行编码。对于4x4亮度块，有九种可选的预测模式；对于16x16亮度块，有四种可选模式；对于色度块，有四种可选预测模式。最终选择使得P和当前块的差值最小的预测模式作为当前块的预测模式。
